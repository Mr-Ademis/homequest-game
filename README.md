# Домашний Квест - Научная работа

## Тема: «Учеба в игре: Секреты Duolingo»

### Описание
Экспериментальное веб-приложение, демонстрирующее принципы геймификации на примере Duolingo.

### Цель
Создать систему мотивации для домашних обязанностей с использованием игровых элементов.

### Игровые механизмы:
1. Балльная система
2. Ежедневная серия (streak)
3. Система достижений
4. Визуальные уведомления
5. Прогресс-трекинг
